Import databáze
 - Je třeba se připojit do MySQL, a v liště nahoře jít do `Server` -> `Data Import`. Zde zaškrtnout radio-button `Import from Self-Contained File` a vybrat export databáze (ten se nachází ve složce sql jako export.sql).
   Ještě se ujistit, že je zvolený `Dump Structure and Data` a dát `Start Import`.
