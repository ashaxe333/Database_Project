Import databáze
 - Je třeba se připojit do MySQL, a v liště nahoře jít do `Server` -> `Data Import`. Zde zaškrtnout radio-button `Import from Self-Contained File` a vybrat export databáze (ten se nachází ve složce sql jako export.sql). Ještě se ujistit, že je zvolený `Dump Structure and Data` a dát `Start Import`.

Připojení k databázi
- Program se musí stáhnout (https://github.com/ashaxe333/Database_Project/releases/tag/v1.0) a před jeho spuštěním bude třeba ještě nastavit soubor `WindowsFormsApp1.exe.config`. Tam upravit uživatelské jméno/heslo/... pro pipojení k databázi
